## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdallah-Ehab/sleep-tracker.git
   ```

2. Navigate to the project directory:
   ```bash
   cd sleep-tracker
   ```

3. Fetch the project dependencies:
   ```bash
   flutter pub get
   ```

4. Run the project:
   ```bash
   flutter run
   ```
